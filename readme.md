# html-decoder-cli

> Decodes html thought the command line

## Install

```sh
cd path/to/html-decoder-cli
```

```sh
npm link
```

## Usage

```sh
html-decoder $'&lt;span&gt;Hello&lt;/span&gt;'
// => '<span>Hello</span>'
```

## Dependencies

html-decoder to be placed in parent directory.
